# UI Style Rules

## Core style
- Desktop-first.
- Dark theme by default.
- Dense but readable.
- Product workspace first.
- Marketing style second.
- Serious percussive fingerstyle workstation feel.

## Avoid
- Gradient blobs.
- Glowing cards.
- Purple futuristic AI look.
- Overly rounded soft SaaS cards.
- Empty decorative sections.
- Giant hero text inside the actual app.

## Prefer
- Left-aligned labels.
- Clear panel layout.
- Tight sync indicators.
- Visible confidence colors.
- Fast access to loop, speed, compare, tuning, capo, and correction tools.
- Visible percussion markers without clutter.
- Distinct user-edited state.

## Visual hierarchy
- Video is important.
- Tab and notation are central.
- Fretboard is supporting but always available.
- Timeline is critical for trust and loop control.
- Percussion markers should feel integrated, not separate from the main lane.
