# Screen Map

## Screen list
- Landing / Home.
- Upload / Input Setup.
- Analysis Progress.
- Main Result Workspace.
- Note Correction Editor.
- Practice Mode.
- Technique and Guitar Intelligence Panel.
- Settings / Export / Project Info.

## Screen goals

### Landing / Home
Explain the app as a percussive fingerstyle guitar transcription and practice workstation.

### Upload / Input Setup
Collect source video and analysis configuration for fingerstyle and basic percussion-aware transcription.

### Analysis Progress
Show visible multi-step pipeline so the user understands audio, timing, fretboard, and percussion-marker processing.

### Main Result Workspace
Review synced output with one main notation/tab lane, technique labels, and percussion markers.

### Note Correction Editor
Fix wrong notes, strings, frets, timing, techniques, and basic percussion markers.

### Practice Mode
Loop and train selected bars with reduced visual distraction and optional focus on percussion-marked passages.

### Technique and Guitar Intelligence Panel
Inspect chords, scales, tuning, capo, fingering, and technique summaries, including basic percussion event summary.

### Settings / Export / Project Info
Control defaults, display options, guitar setup, analysis sensitivity, and exports.

## Navigation logic
- Upload leads to Analysis.
- Analysis leads to Workspace.
- Workspace opens Correction and Practice as dedicated modes.
- Intelligence can be side panel or separate view.
- Settings available globally from top toolbar.

## V1 screen rule
V1 uses one main notation/tab lane.
Percussive events appear as markers and labels, not as a fully separate lane system.
