# Feature Map

## Product identity
- Percussive fingerstyle guitar transcription platform.
- Review and practice workstation.
- Video-first analysis with guitar-specific output.

## Core practice
- Metronome with visual countdown.
- Loop any bar, phrase, or section.
- Speed trainer.
- Bar-by-bar playback.
- Slow motion without pitch shift.
- Focus mode.
- Autoscroll.
- Bookmark difficult bars.
- A/B compare original audio vs rendered tab audio.

## Visual analysis
- Live guitar fretboard.
- Left-hand position replay.
- Hand-position timeline.
- Tab + standard notation.
- Confidence highlights.
- Video-frame scrubbing.
- Live animation synced to tab.
- Percussion markers on the main notation and timeline.

## Guitar intelligence
- Chord detection.
- Alternate voicings.
- Scale detection.
- Root-note highlight.
- Tuning support.
- Capo support.
- Position optimization.
- Transpose.

## Technique detection
### V1
- Hammer-on.
- Pull-off.
- Slide.
- Basic percussion markers.
- Natural harmonic marker support when clearly detected.

### V1.5
- Bend.
- Vibrato.
- Palm mute.
- Let-ring.
- Better harmonic support.
- Better percussion classification.

### Later research
- Artificial harmonics.
- Grace notes.
- Ghost notes.
- Dead notes.
- Right-hand fingering suggestions.
- Hand attribution for percussion.
- Strike-zone detection.
- Contact-type classification.

## Trust layer
- Confidence colors.
- Low-confidence review mode.
- Alternate transcription suggestions.
- Original audio vs rendered audio compare.
- Warnings for tuning and capo uncertainty.
- Clear label for experimental percussion inference.

## Correction layer
- Edit note pitch.
- Edit string.
- Edit fret.
- Edit timing.
- Edit technique tags.
- Add or remove percussion markers.
- Choose alternate suggestion.
- Undo and redo.
- Save correction history.

## Advanced percussive layer
- Kickdrum hit detection.
- Snare hit detection.
- Rimshot hit detection.
- Strike zone detection.
- Hand attribution.
- Body-hit map overlay.
- Contact-type classification.
- Backbeat engine.
- Thumb-slap bass coupling.
- Gesture family clustering.
