# V1 Scope

## In V1

- Video upload.
- Transcription mode selector.
- Audio engine which can recognize notes and chords.
- Tuning support.
- Capo support.
- Synced tablature and standard notation.
- 2D live fretboard replay.
- Confidence coloring for note certainty.
- Manual correction mode.
- A/B compare original audio vs rendered tab audio.
- Looping.
- Slowdown without pitch shift.
- Metronome with visual countdown.
- Speed trainer.
- Focus mode for selected bars.
- Alternate suggestions for low-confidence bars.
- Technique support for hammer-on, pull-off, slide.

## Not in V1

- Full percussion intelligence.
- Body-hit map overlay.
- Contact-type classification.
- Cross-hand coordination timeline.
- Finger independence tracker.
- Two-hand tapping chord notation.
- Gesture family clustering.
- Full artificial harmonic intelligence.
- Detailed hand-attribution engine for every percussion event.

## V1 input restrictions

- Optimize for solo fingerstyle guitar first.
- Visible fretboard preferred with 22 frets.
- Clear audio preferred.
- Short to medium length videos first.
- Alternate tuning and capo supported, but flagged clearly when uncertain.

## V1 success condition

The user can upload a guitar video, inspect synced output, fix mistakes quickly, and practice difficult sections inside the same workspace.
