# Project Overview

## Product

Video to Guitar Tab is a desktop-first web app that converts a guitar performance video into synced guitar tablature, standard percussive and notation, fretboard replay, technique labels, confidence markers, and practice tools.

## Goal

Turn a guitar video into a usable transcription workspace where the user can review, correct, and practice the result.

## Core truth

This is not a generic AI music app or a simple transcription service.
This is a advance Percussive fingersyle guitar transcription, review, and practice workstation.

## Main promise

- User uploads a guitar video.
- System analyzes audio and video.
- System outputs synced tab, notation,bass; melody; harmmony; chords & percussive techniques, fretboard positions, confidence markers, and practice controls.
- User can correct/adjust mistakes directly.
- User practices bars using loop, slowdown, metronome, and focus mode.

## Primary users

- Hobbiest Guitar learners.
- Percussive Fingerstyle players.
- Percussive acoustic players.
- Transcription-focused guitar users.

## What it is not

- Not a DAW.
- Not a generic song recognizer.
- Not a flashy AI SaaS landing page.
- Not a full music production tool.

## Product pillars

- Sync between video, audio, tab, notation,Percussive, bass, melody, harmony, waveform, and fretboard.
- Honest confidence display.
- Fast correction workflow.
- Strong practice workflow.
- Guitar-specific logic like tuning, capo, and fingering position.
