# Technique Taxonomy

## V1 techniques
- Hammer-on.
- Pull-off.
- Slide.
- Basic percussion marker.
- Natural harmonic marker when confidence is high.

## V1.5 techniques
- Bend.
- Vibrato.
- Palm mute.
- Let-ring.
- Better harmonic support.
- Kick-like hit marker.
- Snare-like hit marker.
- Rimshot-like hit marker.

## Later expressive techniques
- Artificial harmonics.
- Grace notes.
- Ghost notes.
- Dead notes.
- Right-hand fingering suggestions.

## Later percussive techniques
- Strike-zone detection.
- Hand attribution.
- Contact-type classification.
- Thumb-slap bass coupling.
- Backbeat engine.
- Gesture family clustering.

## Annotation rules
- Each technique event needs start time, end time, label, and confidence.
- If uncertain, show technique as suggested, not confirmed.
- Percussion in V1 can exist as a marker without full subtype certainty.
- Bar-level technique summaries should not hide note-level detail.
