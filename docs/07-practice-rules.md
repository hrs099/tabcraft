# Practice Rules

## Core controls
- Play.
- Pause.
- Previous bar.
- Next bar.
- Loop current bar.
- Loop selected range.
- Slowdown.
- Metronome.
- Visual countdown.
- Speed trainer.

## Practice behavior
- Looping should be draggable from note lane or bottom timeline.
- Playback slowdown should preserve pitch.
- Speed trainer should raise tempo only after loop completion rule.
- Focus mode hides non-selected bars.
- Autoscroll follows playhead.
- Bookmarks mark problem bars on timeline.
- Percussion-marked bars should be easy to isolate for repetition.

## Practice views
- Large notation / tab area.
- Optional smaller video.
- Bottom loop timeline.
- Right sidebar for trainer controls.
- Visible technique and percussion markers for the selected passage.

## Audio modes
- Original audio.
- Rendered tab audio.
- Quick A/B compare.

## Percussive fingerstyle practice rule
The practice experience should help the user repeat passages where melody, bass, and body-hit rhythm happen together, even if V1 does not split them into separate lanes.

## Success metric
The user can isolate a passage, repeat it, slow it down, and build it back to speed without leaving the app.
