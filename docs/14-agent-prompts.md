# Agent Prompts

## Master instruction
Build a desktop-first web app called Video to Guitar Tab. Position it as a percussive fingerstyle guitar transcription, review, and practice workstation. Prioritize sync between video, tab, notation, waveform, fretboard, and basic percussion markers. Build with mock data first. Do not start with AI models.

## Delivery order
1. Text wireframe plan.
2. Component list.
3. Data schema.
4. Mock demo data.
5. UI for all screens.
6. Playback sync logic.
7. Correction workflow.
8. Practice workflow.
9. Settings and export.
10. Testing hooks.

## Core prompts to keep
- Build main workspace first.
- Add note correction editor.
- Add practice mode with loop, slowdown, metronome, and speed trainer.
- Add confidence-based review mode.
- Add tuning and capo controls.
- Add basic percussion markers in workspace and timeline.

## Rule
Always return wireframe plan in text before generating final UI.
