# Correction Rules

## Editable note fields
- Pitch.
- String.
- Fret.
- Start time.
- End time.
- Duration.
- Technique tags.
- Left-hand finger placeholder.
- Right-hand finger placeholder.
- Velocity placeholder.

## Editable percussion fields in V1
- Percussion marker present or removed.
- Percussion timing.
- Percussion label placeholder.
- Confidence override.

## Correction actions
- Change note pitch.
- Reassign string and fret.
- Merge notes.
- Split notes.
- Add or remove hammer-on.
- Add or remove pull-off.
- Add or remove slide.
- Add or remove percussion marker.
- Select alternate suggestion.
- Revert to AI version.

## Workflow rules
- Selecting a note opens inspector controls.
- Selecting a percussion marker opens percussion edit controls.
- Audio compare is available at note and bar level.
- Undo and redo must always be visible.
- Every save writes correction history.
- User changes should improve trust, not hide original AI state.

## Review tools
- Show only low-confidence notes.
- Show only technique notes.
- Show only corrected notes.
- Show only uncertain bars.
- Show only percussion markers.
