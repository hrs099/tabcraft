# How to Install Python to Run Astra Backend

We have upgraded the TabCraft pipeline to use a state-of-the-art Python AI backend. To run this, you need to install Python on your machine.
Don't worry, it's very easy! Follow these step-by-step instructions.

### Step 1: Download Python
1. Go to the official Python download page: https://www.python.org/downloads/
2. Click the big yellow button that says **"Download Python 3.x.x"** (e.g., Python 3.11 or 3.12).
3. Wait for the installer (`.exe` file) to finish downloading.

### Step 2: Install Python
1. Locate the downloaded file (usually in your `Downloads` folder) and double-click it to run the installer.
2. **CRITICAL STEP:** At the very bottom of the installer window, you will see a checkbox that says:
   **☑ Add Python 3.x to PATH**
   Make absolutely sure this box is **CHECKED** before making any other clicks.
3. Now, click **"Install Now"** at the top of the installer window.
4. If Windows asks "Do you want to allow this app to make changes to your device?", click **"Yes"**.
5. Wait for the progress bar to finish.
6. Once it says "Setup was successful", click **"Close"**.

### Step 3: Run the Astra Backend
That's it! Python is now installed!
To start the AI backend engine:
1. Open the project folder on your computer.
2. Go into the `server` folder.
3. Double-click the file named `start.bat`.
4. First time taking longer is normal! It will automatically create a virtual environment and download the AI models required. Once it says `Application startup complete`, it is running and ready!
