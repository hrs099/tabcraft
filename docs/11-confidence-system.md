# Confidence System

## Confidence sources
- Audio confidence.
- Video confidence.
- Fused confidence.

## Visual colors
- Green = confident.
- Yellow = uncertain.
- Red = likely wrong.
- Blue = user corrected.

## Rules
- Confidence must be visible directly in workspace.
- Uncertain bars should expose alternate suggestions.
- Low-confidence mode should filter fast.
- Confidence should exist at note, bar, technique, tuning, capo, and percussion-marker level.

## Trust rules
- Never hide uncertainty.
- Never silently replace user edits with AI edits.
- Mark experimental detections clearly.
- Keep original AI result accessible after correction.
- Percussion markers with weak evidence should be shown as tentative, not certain.
